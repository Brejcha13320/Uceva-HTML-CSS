# 📘 Guía de HTML Básico

Este repositorio contiene una **guía práctica de HTML** pensada para estudiantes que inician en el desarrollo web.  
El objetivo es aprender paso a paso las etiquetas más importantes mediante ejemplos simples y ejercicios aplicados.

---

## 🚀 Contenido

La guía está organizada en **módulos temáticos**, cada uno con:
- Explicación breve de las etiquetas.
- Ejemplos comentados en HTML.
- Un ejercicio práctico.

Los temas incluyen:
- Estructura básica de un documento HTML.
- Texto y formato.
- Listas y enlaces.
- Imágenes y tablas.
- Formularios (básicos y avanzados).
- Multimedia.
- Etiquetas semánticas.
- Validaciones HTML5.

*(La organización puede variar, pero cada archivo `.html` corresponde a un tema específico.)*

---

## 🛠️ Requisitos

- Editor de texto (ejemplo: [Visual Studio Code](https://code.visualstudio.com/)).  
- Navegador web actualizado (Chrome, Firefox, Edge, etc.).
